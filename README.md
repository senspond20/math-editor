# React ProseMirror Markdown And Math Editor

React ProseMirror Markdown And Math Editor 

> ProseMirror is a useful tool for creating rich text editors in web browsers. I am trying to migrate to the React version.

![2024-05-30-1](./doc/2024-05-30-1.png)


### install

```
npm install
```

### dev

```
npm run dev
```

### build

```
npm run build
```
